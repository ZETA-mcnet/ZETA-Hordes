# ZETA-Hordes

Transform your Minecraft server into a heart-pounding survival experience with ZETA-Hordes, an advanced zombie horde system that brings the thrill of survival horror to your world.

[![Build Status](https://github.com/ZETA-mcnet/ZETA-Hordes/workflows/Build%20and%20Release/badge.svg)](https://github.com/ZETA-mcnet/ZETA-Hordes/actions)
[![Latest Release](https://img.shields.io/github/v/release/ZETA-mcnet/ZETA-Hordes)](https://github.com/ZETA-mcnet/ZETA-Hordes/releases)
[![Paper Compatible](https://img.shields.io/badge/Paper-1.21.4-brightgreen.svg)](https://papermc.io/)

## 🧟 Features

- **Dynamic Horde System**: Intelligent zombie hordes that hunt players with advanced pathfinding
- **Custom Zombie Types**: Unique zombie variants with special abilities and attributes
- **Adaptive Difficulty**: Smart scaling based on player count and activity
- **Anti-AFK System**: Sophisticated detection to prevent AFK farming while maintaining fair gameplay
- **Configurable Settings**: Extensive configuration options for:
  - Horde size and composition
  - Spawn timing and locations
  - Custom zombie abilities
  - Player targeting preferences
  - World-specific settings

## 📋 Requirements

- Paper/Spigot Server 1.18.X - 1.21.X (1.21.4 recommended)
- Java 21 or higher
- Minimum 2GB server RAM (4GB recommended)

## ⚙️ Installation

1. Download the latest release from our [Releases page](https://github.com/ZETA-mcnet/ZETA-Hordes/releases)
2. Place the JAR file in your server's `plugins` folder
3. Restart your server
4. Configure the plugin in `plugins/ZETA-Hordes/config.yml`

## 🛠️ Building from Source

```bash
# Clone the repository
git clone https://github.com/ZETA-mcnet/ZETA-Hordes.git
cd ZETA-Hordes

# Build with Gradle
./gradlew clean build
```

The compiled JAR will be in `build/libs/`.
